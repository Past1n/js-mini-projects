# JavaScript Interactive Components Collection

A professional portfolio showcasing interactive web applications built with **Vanilla JavaScript**. This repository demonstrates proficiency in DOM manipulation, event-driven programming, and modern CSS3 techniques.

## 🚀 Projects Overview

### 1. 🎯 Aim Trainer
A precision-based reaction game designed to test and improve clicking speed and accuracy. 
- **Technical Focus:** State management, asynchronous timing functions (`setInterval`), and dynamic coordinate calculation using `getBoundingClientRect`.
- **Key Features:** 
  - Multiple session durations.
  - Randomized target generation (size, position, and color).
  - Real-time score tracking and game-over state handling.

### 2. ✨ Interactive Board
A dynamic sensory grid that responds to user movement with fluid visual feedback.
- **Technical Focus:** Event delegation, high-performance DOM rendering (400+ elements), and advanced CSS transitions.
- **Key Features:**
  - Neon hover effects with randomized color palettes.
  - Smooth "fade-out" animations using persistent CSS transitions.
  - Responsive layout for different screen sizes.
